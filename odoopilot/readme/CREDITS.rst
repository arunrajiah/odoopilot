OdooPilot is built and maintained by a solo developer. If it saves
your team time, please consider sponsoring at
`github.com/sponsors/arunrajiah
<https://github.com/sponsors/arunrajiah>`_.

The product builds on:

* `Anthropic Claude <https://www.anthropic.com>`_ for the default
  LLM backend.
* `OpenAI <https://openai.com>`_ for an alternative LLM and the
  Whisper STT model.
* `Groq <https://groq.com>`_ for the fastest free-tier inference of
  both Llama and Whisper.
* `Ollama <https://ollama.com>`_ for the 100%-local self-hosted
  option.
* `Telegram Bot API <https://core.telegram.org/bots/api>`_ and
  `Meta WhatsApp Cloud API
  <https://developers.facebook.com/docs/whatsapp/cloud-api>`_ for
  the messaging surface.
