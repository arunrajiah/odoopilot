What this addon does

This Odoo addon is the companion piece to the OdooPilot service (a self-hosted FastAPI app). The addon handles:

  • One-click account linking — employees send /link to the Telegram bot and click a magic link inside Odoo to authenticate.
  • Settings page — configure the OdooPilot service URL and toggle channels directly from Settings → Technical → OdooPilot.
  • Audit log — every AI action is recorded: who asked, which tool ran, success/failure.
  • Identity store — maps Telegram chat IDs to Odoo users securely.

The default service URL points to odoopilot.fly.dev — a free shared instance good for evaluation. For production, self-host the OdooPilot service on Fly.io (free tier) or any server.

Quick setup

  1. Install this addon from the App Store.
  2. Go to Settings → Technical → OdooPilot and verify the service URL.
  3. Create a Telegram bot via @BotFather and add the token to your OdooPilot service.
  4. Tell employees to send /link to the bot — done in 30 seconds.

No Odoo Enterprise required. OdooPilot works entirely on Odoo 17 Community Edition.
