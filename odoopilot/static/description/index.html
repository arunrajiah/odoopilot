Everything runs inside Odoo

Unlike other AI integrations, OdooPilot has no external service to host. The Telegram webhook, the LLM calls, and the Odoo tools all live inside this single addon. Install it, configure your keys, and you're live.

🔌 Webhook controller

Receives Telegram updates directly inside Odoo HTTP.

🤖 LLM loop

Calls Anthropic, OpenAI, or Groq — your key, no middleman.

🔐 Identity linking

One-click magic link maps each Telegram user to an Odoo employee.

📝 Audit trail

Every AI action is logged — who asked, what ran, success or failure.

No Odoo Enterprise required. Works entirely on Odoo 17 Community Edition. Supports Anthropic Claude, OpenAI GPT, and Groq.

Up and running in 5 minutes

  1. Install this addon from the Odoo App Store.
  2. Create a Telegram bot via @BotFather — copy the token.
  3. Get an API key from Anthropic, OpenAI, or Groq (free tier works).
  4. Go to Settings → OdooPilot, paste both keys, choose your LLM, click Register Webhook.
  5. Tell each team member to message the bot — they tap a link to link their Odoo account. Done.
